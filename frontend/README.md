# Frontend Application

This is the frontend application for the Full-Stack Todo application, built with Next.js, React, and TypeScript.

## Features

- User authentication (signup, login, logout)
- Todo management (create, read, update, delete, toggle completion)

## Setup

1.  **Install dependencies**:
    ```bash
    cd frontend
    npm install
    ```

2.  **Create a `.env.local` file**:
    Copy `frontend/.env.local.example` to `frontend/.env.local` and update the values.

3.  **Run the application**:
    ```bash
    npm run dev
    ```

The application will be accessible at `http://localhost:3000`.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run build`

Builds the application for production.

### `npm run start`

Runs the built application in production mode.

### `npm run lint`

Runs ESLint to check for code quality issues.
